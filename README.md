ScribeNotes is a web application that converts lecture and lab audio recordings into structured study notes using ElevenLabs Scribe v2.

## What it does
- Upload lecture audio
- Transcribe speech using Scribe v2
- Extract key points and questions
- Generate summaries
- Export notes as a PDF

## Tech Stack
- FastAPI (Python)
- ElevenLabs Scribe v2
- HTML, CSS, JavaScript

## Hackathon
Built for the Scribe v2 Hackathon to demonstrate real-world transcription usage.
